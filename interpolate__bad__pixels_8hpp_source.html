38 /**
39  * If the potential "bad" pixel value deviates more than this "factor" from the mean
40  * of the surrounding pixels, then the pixel is removed, i.e. interpolated by the mean

44  * @param Filter core size (N x N), odd number > 1, allowed values are 3,5,7,9

47  * @return Image with interpolated bad pixels (unique_ptr<cimg_library::CImg<ImageType>>)
48  *
49  *
50  * TODO: Move the actual algorithm out of this pipeline function to the "algorithms" and also write
51  * unit tests for this filter independent from this pipeline functionality.

53  * TODO: Add:

55  */

57 auto interpolate_bad_pixels(

59  unsigned int filter_core_size = 3,
60  starmathpp::algorithm::BadPixelMedianInterpolator::ThresholdDirection::TypeE threshold_direction =
61  starmathpp::algorithm::BadPixelMedianInterpolator::ThresholdDirection::BOTH) {
62  return ranges::views::transform(
