Detailed Description

InterpolationTypeT specifies the method which should be used to interpolate during the scaling process.

https://cimg.eu/reference/structcimg__library_1_1CImg.html resize ( const int size_x, const int size_y = -100, const int size_z = -100, const int size_c = -100, const int interpolation_type = 1 ...

interpolation_type Method of interpolation:

-1 = no interpolation: raw memory resizing. 0 = no interpolation: additional space is filled according to boundary_conditions. 1 = nearest-neighbor interpolation. 2 = moving average interpolation. 3 = linear interpolation. 4 = grid interpolation. 5 = cubic interpolation. 6 = lanczos interpolation.
