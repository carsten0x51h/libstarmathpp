Detailed Description

template<class T, typename>
class starmathpp::Rect< T, typename >

Pre-declare the template class itself to be ale to declare operator<<. Allow only numeric types for Rect template. See https://stackoverflow.com/questions/14294267/class-template-for-numeric-types

Rect structure (X x Y x W x H).
